本教程将以 T2 开发板为基础，介绍 Tuya OS 的开发流程和技能。首先大家要有一个基本的认知:

::: note Tuya OS 它是开发版和Tuya 开发者平台绑定的，开发板的所有精髓是接入Tuya 开发者平台。
:::
