## Windows

- 驱动下载：👉[CH343SER_EXE](https://file.wch.cn/download/file?id=315)
- 双击打开 `CH343SER.EXE` 程序
- 点击安装即可：

<center>

![alt text](/img/bruntools/驱动安装.png)
</center>

- 查看设备端口

::: tip 在电脑的设备管理器中，即可看到烧录架的端口号，默认名称：`USB-Enhanced-SERIAL CH343`
:::

## Linux (Ubuntu20.04 为例) {#LinuxDriver}

以 Ubuntu 为例，Ubuntu 不需要自行安装驱动，烧录架插进去之后，会自动识别成 ttyACM 设备，可以使用以下指令查看：
```bash
ls /dev/ttyACM*
```
- 运行结果：

<center>

![alt text](/img/bruntools/linux_install.png)
</center>

#### 关于权限问题

::: warning  第一次可能遇到权限不足的情况，导致无法使用ttyACM串口，使用以下指令授权即可解决：

``` shell
sudo chmod 777 /dev/ttyACM0  
```
``` shell
sudo usermod -a -G dialout $USER
```

:::

::: note 说明：

可能你的烧录架识别到的不是`ttyACM0` 而是 `ttyACM1` 或者数字更高的`ACM`，相应的改成正确的就行。
 `sudo usermod -a -G dialout $USER` 是把当前用户的权限组添加到 `dialout`。以后每次使用就不需要重新授权。
:::
