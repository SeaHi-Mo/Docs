## 什么是 Hermes Agent？

它不是 tethered 到 IDE 的编程助手，也不是围绕单个 API 的聊天机器人包装器。它是一个**自主智能体**，运行时间越长能力越强。它可以部署在任何地方 —— 5 美元的 VPS、GPU 集群，或几乎空闲时无成本的无服务器基础设施（Daytona、Modal）。通过 Telegram 与它交流，而它在你从未 SSH 登录过的云虚拟机上工作。它不受限于你的笔记本电脑。

## 主要功能

- **闭环学习** —— 智能体策划的记忆，定期提醒，自主技能创建，使用过程中的技能自我改进，FTS5 跨会话召回与 LLM 摘要，以及 [Honcho](https://github.com/plastic-labs/honcho) 辩证用户建模
- **随处运行，不限于笔记本电脑** —— 6 种终端后端：本地、Docker、SSH、Daytona、Singularity、Modal。Daytona 和 Modal 提供无服务器持久化 —— 空闲时环境休眠，成本几乎为零
- **与您同在** —— CLI、Telegram、Discord、Slack、WhatsApp、Signal、Matrix、Mattermost、Email、SMS、DingTalk、Feishu、WeCom、BlueBubbles、Home Assistant —— 一个网关支持 15+ 平台
- **由模型训练者构建** —— 由 [Nous Research](https://nousresearch.com) 创建，该实验室开发了 Hermes、Nomos 和 Psyche。适用于 [Nous Portal](https://portal.nousresearch.com)、[OpenRouter](https://openrouter.ai)、OpenAI 或任何端点
- **定时自动化** —— 内置 cron，可投递到任何平台
- **委托与并行化** —— 生成隔离的子智能体进行并行工作流。通过 `execute_code` 实现程序化工具调用，将多步骤管道压缩为单次推理调用
- **开放标准技能** —— 兼容 [agentskills.io](https://agentskills.io)。技能可移植、可共享，并通过技能中心社区贡献
- **完整网页控制** —— 搜索、提取、浏览、视觉、图像生成、TTS
- **MCP 支持** —— 连接到任何 MCP 服务器以扩展工具能力
- **研究就绪** —— 批处理、轨迹导出、使用 Atropos 进行 RL 训练。由 [Nous Research](https://nousresearch.com) 构建 —— Hermes、Nomos 和 Psyche 模型背后的实验室
